# Skill Backlog

Use this file for maintenance items that are not mature enough to become a repo change yet.

Record entries like this:

```text
- Date:
  Challenge:
  Gap:
  Evidence:
  Proposed action:
  Why backlog only:
```

Rules:

- Prefer backlog over a new skill when the pattern appeared only once.
- Promote an entry only after the pattern is repeated or the missing workflow becomes stable.
- When an item is promoted into a script, lab, test, or skill update, remove or strike the backlog entry.
